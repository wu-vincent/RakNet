RakNet is a high-performance network library designed for games and other latency-sensitive applications. It provides reliable and unreliable data transfer over UDP, with a rich set of plugins for common multiplayer game features.

## Key Features

- **Reliable UDP** — multiple reliability and ordering modes for different data types
- **NAT Punchthrough** — connect peer-to-peer users behind routers without port forwarding
- **Replica Manager** — automatic object replication, creation, destruction, and serialization
- **Autopatcher** — delta-compressed patching system for game updates
- **Voice Chat** — integrated voice communication via RakVoice
- **Fully Connected Mesh** — peer-to-peer host determination and migration
- **Secure Connections** — encrypted communication using SLNet's security layer
- **Cross-Platform** — Windows, Linux, macOS, iOS, Android
