<Callout type="warn">
These training videos were originally created as Flash-based (SWF) or Windows Media (WMV) screencasts using Camtasia Studio. The Flash-based videos are no longer playable in modern browsers since Flash was discontinued in 2020. They are listed here for historical reference.
</Callout>

The following video tutorials were produced by Kevin Jenkins to demonstrate various RakNet features:

### Hello World / RakNet Introduction

- **File:** `RakNetIntroVideo.swf`
- **Format:** Flash (SWF), 800x618
- **Date:** February 2008
- **Description:** An introductory video covering the basics of getting started with RakNet.

### Data Replication

- **File:** `DataReplication.wmv`
- **Format:** Windows Media Video (WMV), 640x580
- **Description:** Covers the data replication system used for synchronizing game objects across the network.

### RPC3 (Remote Procedure Calls)

- **File:** `RPC3.swf`
- **Format:** Flash (SWF), 800x618
- **Date:** July 2008
- **Description:** Demonstrates the RPC3 system for calling remote procedures across the network. See also [RPC4](/plugins/rpc4).

### Autopatcher

- **File:** `AutopatcherDemo.swf`
- **Format:** Flash (SWF), 800x618
- **Date:** July 2008
- **Description:** Demonstrates the [Autopatcher](/plugins/autopatcher) plugin for automatic game patching.

### ReplicaManager3

- **File:** `ReplicaManager3Video.swf`
- **Format:** Flash (SWF), 1024x786
- **Date:** June 2009
- **Description:** Demonstrates the [ReplicaManager3](/plugins/replica-manager3) plugin for automatic creation, destruction, and serialization of game objects.

### RakNet Major Features Overview

- **File:** `RakNetMajorFeatures.swf`
- **Format:** Flash (SWF), 1024x786
- **Date:** May 2009
- **Description:** A comprehensive overview of RakNet's major features and capabilities.
